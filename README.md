========================== Simple ecommerce  application ==============

This application shows list of products on page its kind of small ecommerse app

-----pages------
HOME
LOGIN
CART
ORDER SUMMURY

----operations---

add item to cart
Cart page where you can increase or decrese product Qty
User authentication for login based on Json web token
User Logout functionality
Cart qty 

----Techonology used------
    React
    Redux
    Redux thunk (optional)
    Javascript(es6)
    CSS3
    Material UI (for CSS and styling component)
    ESlint

-------INSTALLATION---------
1 . Clone the code from repo (https://github.com/hanamantk/ecommerce-app.git)
    git clone https://github.com/hanamantk/ecommerce-app.git
    
2. navigate to ecommerce-app project directory in local machine
3. make sure node and npm are updated in your machine (node 12+ version, npm 6+ version)
3. npm install
4. npm start
5. project will be opened in 3000 port in browser(http://localhost:3000/)
